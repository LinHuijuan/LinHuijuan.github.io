  • 浮动技术允许元素浮动到另外一个元素的左侧或右侧，而不是默认的一个堆叠另一个。float 的主要用途是布置出多个列并且浮动文字以环绕图片。下面我们快速浏览一下 float属性并用起来。
    • float 属性有四个可能的值：
      • left — 将元素浮动到左侧。
      • right — 将元素浮动到右侧。
      • none — 默认值, 不浮动。
      • inherit — 继承父元素的浮动属性。

# 2.2 清除浮动

有的时候浮动是需要清除的，比如说，我需要指定一个元素必须移动到在它之前的浮动元素下面。

![清除浮动](..\img\layout 5.jpg)

但实h际上，我让色块1float: left，色块2float: right后，色块1和2会脱离文档流，此时色块3就会往上移，被色块1和2覆盖。

![无清除浮动](..\img\layout 6.jpg)

这个时候clear属性就要派上用场了！

  • clear CSS (opens new window) 属性指定一个元素是否必须移动(清除浮动后)到在它之前的浮动元素下面。clear属性适用于浮动和非浮动元素。
  • none：元素不会向下移动清除之前的浮动。（默认值）
  • left：元素被向下移动用于清除之前的左浮动。
  • right：元素被向下移动用于清除之前的右浮动。
  • both：元素被向下移动用于清除之前的左右浮动。

显然，要想色块3被向下移动，我们只需要给它添加样式clear: both;。

# 2.3 clearfix

注意：如果一个元素里只有浮动元素，那它的高度会是0。如果你想要它自适应即包含所有浮动元素，那你需要让它能自己清除浮动。这种方法叫做clearfix，即使用 ::after伪元素设置clear属性作为替代。

// ::after指伪元素，相当于元素的子元素 <span></span>（inline行内元素）。
#container::after { 
  content: "";	
  display: block; 
  clear: both;
}
// 相当于给这个元素增加了一个内容为透明的span元素，来撑开元素的高

打印得到的也是a的地址值。

  • 可以把存放一个int类型变量的地址赋给一个int *指针类型的变量。

  • '='右边是&a，a是int类型的变量数值5，&是取地址符，&a就是拿到int类型a的数值的地址。

# 3.1 typedef

使用typedef定义结构体同时，给stu结构体别名STU，后续定义可不用使用struct stu，直接使用STU即可。

typedef struct stu
{
    char  aa;
    short bb;
    int   cc;
}STU;

STU stu1;

# 四、库函数

# 4.1 stdlib.h

stdlib.h里面定义了五种类型、一些宏和通用工具函数。

变量类型例如size_t、wchar_t、div_t、ldiv_t和lldiv_t；

宏例如NULL、EXIT_FAILURE、EXIT_SUCCESS、RAND_MAX和MB_CUR_MAX等等；

常用的函数如malloc()、calloc()、realloc()、free()、system()、atoi()、atol()、rand()、srand()、exit()等等。

  • size_t：这是无符号整数类型，它是 sizeof 关键字的结果。
  • NULL：这个宏是一个空指针常量的值。

# 4.1.1 malloc()

  • 分配所需的内存空间，并返回一个指向它的指针。
// 动态分配存储空间
void *malloc(size_t size)
1
2

其中size是指内存大小，以字节（Byte）为单位。若请求失败，则返回NULL。

# 4.2 assert.h

用于验证程序做出的假设，并在假设为假时输出诊断消息。

printf("请输入矩阵的行和列(行列必须相同)\n");
scanf("%d %d", &m, &n);
assert(m == n);
1
2
3

如果表达式为 FALSE (0), 程序将报告错误，并终止执行。如果表达式不为 0，则继续执行后面的语句。这个宏通常原来判断程序中是否出现了明显非法的数据，如果出现了终止程序以免导致严重后果，同时也便于查找错误。

# 7.1 基于TextField实现顶部SearchBar

参考教程：https://www.cnblogs.com/wangjunwei/p/12213272.html

class _SearchBarState extends State<SearchBar> {
    // 是否显示 × 按钮
    bool showClear = false;
    // 获取搜索框内文字的变化
    final TextEditingController _controller = TextEditingController();

    @override
    void initState() {
        // 要实现的效果是语音输入后，把文字带到搜索页的文本框上。所以就需要defaultText来设置默认的文字。
        if (widget.defaultText != null) {
            setState(() {
                _controller.text = widget.defaultText;
            });
        }
        super.initState();
    }

    @override
    Widget build(BuildContext context) {
        // 根据当前输入框的不同类型样式，设置不同的样式。
        return widget.searchBarType == SearchBarType.normal
            ? _genNormalSearch()
            : _genHomeSearch();
    }
}

# 12.1 更新Flutter SDK 和 packages

  • 在项目的根目录下执行命令
flutter upgrade
1

flutter sdk将会更新到最新版，packages会更新到所能兼容的最新版。

如果不翻墙的话，需要添加镜像。

# 12.2 只更新包packages

# 12.2.1 手动更新（推荐）

  • https://pub.dartlang.org
  • 在更新之前要看下文档有哪些变更。

# 12.2.2 命令的方式

flutter packages upgrade // 更新
flutter packages get // 安装
